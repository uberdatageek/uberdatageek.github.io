Apache Spark API with Python

Wrote a working WordCount program using Spark API in Python. Downloaded and read “all-bible.txt” file from an HDFS directory and wrote results of calculations to an HDFS file. To improve the word count, removed any punctuations that might have been included in to the words. Also transformed all words into lower case so that case sensitivity did not affect the word count.
